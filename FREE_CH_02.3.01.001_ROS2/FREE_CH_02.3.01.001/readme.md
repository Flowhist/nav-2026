# 富锐单线激光雷达ROS2驱动程序

ROS2驱动程序，从单线激光雷达读取原始数据并将数据作为激光扫描消息发布。驱动程序目前仅支持从源代码构建

### 支持的硬件

支持C20X、C21X、C22X、C26X、C28X及H系列网络通讯雷达。

### 前提

- 搭载Ubantu20.04（22.04）的电脑

- 已安装ROS2

- 连接雷达的电源线和网线

- 设置主机和传感器的以太网，必须位于同一网段中。

- 按照实际情况配置free_lidar_launch.py中的参数


  | 参数项          | 类型   | 举例              | 说明                                      |
  | --------------- | ------ |-----------------|-----------------------------------------|
  | frame_id        | string | scan            | 雷达消息的帧名称                                |
  | is_ethernet | bool | False           | 网口/串口开关，True为网口，False为串口                |
  | scanner_ip      | string | "192.168.1.111" | 雷达IP                                    |
  | port_name | string | "/dev/ttyUSB0"  | 串口端口名                                   |
  | baud | int | 921600          | 串口波特率（115200/460800/921600）             |
  | scan_frequency  | int    | 15              | 扫描频率（15Hz）    （30HZ）                    |
  | scan_resolution | int    | 3333            | 角分辨率（0.3333°，C和H系列的雷达需要做好区分）0.1°为1000   |
  | start_angle     | int    | -45             | 开始角度                                    |
  | stop_angle      | int    | 225             | 停止角度                                    |
  | offset_angle    | int    | 0               | 角度偏移量                                   |
  | filter_switch   | bool   | 0（关）或1（开）            | 拖尾滤波开关                                  |
  | cluster_num | int | 10              | 滤波聚类点数(避免误过滤)                           |
  | broad_filter_num | int | 20              | 展宽过滤点数（高反两侧强制过滤）                        |
  | NOR_switch | int | 1               | 反射率归一化开关，1为开启，0为关闭（原始反射率                |
  | is_reverse_postion | bool | False           | 反装标志,False为正装，True为反装                   |
  | topic_name | string | "/scan2"        | 点云话题名称                                  |
  


### 启动步骤

1. 将驱动文件放在colcon_ws/src下

2. 返回到colcon_ws下，开启终端

3. 输入指令：colcon build

4. 输入指令：source ~/colcon_ws/install/setup.sh

5. 输入指令：ros2 launch free_lidar free_lidar_launch.py
